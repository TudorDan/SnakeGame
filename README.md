# Snake game

This is a snake game with full 360 degree rotation.

## Development

### Inside IntelliJ Idea

   Open the project as a Maven project. (by opening the =pom.xml=)

   Run it from the maven sidebar by running the =javafx:run= target

### From command line

   Build it with

   ``` 
   mvn javafx:compile
```
    

   Run in with

   ```
mvn javafx:run
```
